# Simple Console Calculator (Python Mini-Project)

## Project Title & Description
This project is a simple, command-line calculator built in Python. It is designed to perform basic arithmetic operations (addition, subtraction, multiplication, and division) and serves as a fundamental example of console-based application development. This mini-project was developed using **Visual Studio Code (VS Code)** and is maintained using **Git** and **GitHub**.

---

## Project Features List
The current version of the calculator supports the following features:

* **Addition:** Adds two numbers.
* **Subtraction:** Subtracts the second number from the first.
* **Multiplication:** Multiplies two numbers.
* **Division:** Divides the first number by the second, and includes error handling for division by zero.
